# MfTest Microflow Specification

This document describes the expected behavior of microflows generated by
`02-microflow-examples.mdl`. Each code block is an executable test that validates
the microflow works correctly at runtime.

Run all tests with:
```bash
mxcli test microflow-spec.test.md -p app.mpr
```

## Basic Operations

### M001: Hello World

The simplest microflow — returns a Boolean `true`.

```mdl-test
/** @test Basic return value
 *  @expect $result = true
 */
$result = CALL MICROFLOW MfTest.M001_HelloWorld();
```

### M002: Basic Variables

Declares two Integer variables (10 and 5) and returns their product.

```mdl-test
/** @test Basic arithmetic (10 * 5)
 *  @expect $result = 50
 */
$result = CALL MICROFLOW MfTest.M002_BasicVariables();
```

### M003: String Operations

Concatenates first and last name with a space separator.

```mdl-test
/** @test String concatenation
 *  @expect $result = 'John Doe'
 */
$result = CALL MICROFLOW MfTest.M003_StringOperations(
  FirstName = 'John', LastName = 'Doe'
);
```

### M004: Boolean Logic

Returns the AND of two Boolean parameters.

```mdl-test
/** @test Boolean AND logic
 *  @expect $result = true
 */
$result = CALL MICROFLOW MfTest.M004_BooleanLogic(
  IsActive = true, IsValid = true
);
```

## Control Flow

### M006: Simple IF

Tests a single IF condition — returns a message if Value > 10.

```mdl-test
/** @test Simple IF condition (20 > 10)
 *  @expect $result = 'Value is greater than 10'
 */
$result = CALL MICROFLOW MfTest.M006_SimpleIf(Value = 20);
```

### M007: IF/ELSE

Tests IF/ELSE branching — returns "High" if Value > 100, otherwise "Low".

```mdl-test
/** @test IF/ELSE condition (200 > 100)
 *  @expect $result = 'High'
 */
$result = CALL MICROFLOW MfTest.M007_IfElse(Value = 200);
```

### M008: Nested IF

Tests nested IF for letter grading. Score 85 should return 'B'.

```mdl-test
/** @test Nested IF grading (score 85)
 *  @expect $result = 'B'
 */
$result = CALL MICROFLOW MfTest.M008_NestedIf(Score = 85);
```

### M035: Multiple Return Paths

Tests that different input values produce different return strings.

```mdl-test
/** @test Multiple return paths - negative
 *  @expect $result = 'negative'
 */
$result = CALL MICROFLOW MfTest.M035_MultipleReturnPaths(Value = -5);
```

```mdl-test
/** @test Multiple return paths - zero
 *  @expect $result = 'zero'
 */
$result = CALL MICROFLOW MfTest.M035_MultipleReturnPaths(Value = 0);
```

```mdl-test
/** @test Multiple return paths - positive
 *  @expect $result = 'positive'
 */
$result = CALL MICROFLOW MfTest.M035_MultipleReturnPaths(Value = 42);
```

## Entity Operations

### M012: Create Entity

Creates a Product entity with Name and Code attributes.

```mdl-test
/**
 * @test Entity creation
 * @expect $product/Name = 'TestProduct'
 */
$product = CALL MICROFLOW MfTest.M012_CreateEntity(
  Name = 'TestProduct', Code = 'TP-001'
);
COMMIT $product;
```

## Sub-Microflow Calls

### M038: Call Chain

Tests that microflows can call other microflows and return results.

```mdl-test
/** @test Sub-microflow call chain
 *  @expect $result = true
 */
$result = CALL MICROFLOW MfTest.M038_1_TestCallWithResult();
```
