Enumeration Creation Syntax
===========================

Basic syntax:
  CREATE ENUMERATION Module.EnumName (
    Value1 'Display Caption 1',
    Value2 'Display Caption 2',
    ...
  );

Notes:
  - Value names must be valid identifiers (no spaces, no reserved keywords)
  - Captions are optional but recommended for display
  - Each value should be on its own line

Example:
  CREATE ENUMERATION MyModule.OrderStatus (
    Pending 'Pending Approval',
    Processing 'Being Processed',
    Shipped 'Shipped to Customer',
    Delivered 'Delivered',
    Cancelled 'Order Cancelled'
  );

Using in an entity:
  CREATE PERSISTENT ENTITY MyModule.Order (
    OrderNumber: String(20) NOT NULL,
    OrderStatus: Enumeration(MyModule.OrderStatus)
  );
